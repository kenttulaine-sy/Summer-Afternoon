# Summer Afternoon - Godot Project

A peaceful summer afternoon walk in the countryside.

## Controls
- **WASD** - Walk around
- **Shift** - Run
- **Mouse** - Rotate camera (planned)

## Project Structure
- `scenes/main.tscn` - Main game scene
- `scripts/player.gd` - Player controller with walking/running animations
- `assets/character/` - ReadyPlayerMe character model with textures

## Character
Uses ReadyPlayerMe Cyberpunk avatar with rigged animations.

## Features
- Third-person character controller
- Animated walking/running
- Summer afternoon lighting (warm sun, blue sky)
- Simple ground to walk on

## Setup
1. Open Godot 4.3+
2. Import project from this folder
3. Play the main scene
